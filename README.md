Проект состоит из двух частей: ноутбук с ubuntu и мачта с одноплатным компьютером. Пульт управления подключается к ноутбуку по USB-Uart конвертеру. На ноутбуке поднят utp сервер/клиент который постоянно отправляет на модуль передатчика данные о положении стиков-тумблеров и принимает обратно телеметрию. На ноутбуке находится rtsp-видеопроигрыватель для приёма аналогового видео с видеоприёмника подключенному к CVBS2ETHERNET. На ноутбуке поднимается веб страница с отображение видео, телеметрией с дрона и возможностью настройки видеоприёмника.
